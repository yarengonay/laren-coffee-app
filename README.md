# Laren Cafe App ☕

Bu proje, bilgisayar programcılığı bölümü için geliştirdiğim modern bir kahve dükkanı mobil uygulama arayüzüdür. Jetpack Compose kullanılarak tasarlanmıştır.

## Özellikler
* Şık ve kullanıcı dostu giriş ekranı.
* Kategorilere ayrılmış kahve menüsü.
* Detaylı ürün görselleri ve açıklamaları.
* Sepet ve sipariş yönetim arayüzü.
